# Gestures

Gesture commands also use refs from `snapshot`.

```bash
appium-cli long_press e12
appium-cli double_tap e12
appium-cli drag e12 500 900
appium-cli fling up
appium-cli pinch_open e12
appium-cli pinch_close e12
```

Use gestures only when simpler actions such as `tap`, `type_text`, or `scroll` are insufficient.
